Community

In addition to the official documentation and plugins, below you will find some great community contributions.

  • SEO Azure Crawler - A cloud-based solution for creating HTML snapshots of your Durandal application to serve to google's crawler.
  • dFiddle - A cut down version of Durandal, hosted as a Github repo with only one gh-pages branch. By cutting things down to the bare minimum, dFiddle allows bringing up live Durandal fiddles in minutes.
  • knockout-delegatedEvents - A KO plugin that makes delegated events easy, without resorting to manual jQuery code.
  • Knockout Utility Signatures - A brief gist showing all the KO utility functions and their signatures.
